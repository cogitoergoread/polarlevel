# Polar Pulse meter data with level info
This repository contains a Python script that processes Polar Pulse meter data, including level information. 

## Features
- reads data via an API call [Polar Open AccessLink](https://www.polar.com/accesslink-api/)
- use appropriate OAuth2 authentication
- processes the data to extract relevant information, including level info, geolocation, pulse and timestamps
- outputs the processed data in a structured format (e.g., CSV, JSON)
